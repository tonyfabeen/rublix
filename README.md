rublix
======

LXC bindings in Ruby

## Requirements
- Ubuntu 11.04 / 11.10 64-bit system
- Superuser privileges (required in some scripts)
- Ruby 1.9.x
- Packages : lxc, lxc-dev >= 0.9.0
