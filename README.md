Rublix
======

  LXC bindings in Ruby

## Requirements
  - Ubuntu 11.04 / 11.10 64-bit system
  - Superuser privileges
  - Ruby 1.9.x
  - Ubuntu Packages : lxc, lxc-dev >= 0.9.0
  - RubyGems : Bundler, Sinatra
